# AI Systems That Turn Information Into Action

I build AI systems that help organisations make better decisions, reduce manual workload, and deliver services more efficiently.

My work sits at the intersection of artificial intelligence, healthcare, automation, and real-world operations.

## What I Build

### Decision Support Systems

AI solutions that help prioritise, triage, classify, and route information so the right actions happen faster.

### Intelligent Assistants

LLM-powered applications that help users retrieve information, automate tasks, and improve productivity.

### Computer Vision Applications

Systems that extract actionable insights from images and video, from object detection to automated monitoring.

### Data & Predictive Analytics

Machine learning solutions that transform raw data into forecasts, recommendations, and operational intelligence.

### AI-Driven Automation

Tools that reduce repetitive work, streamline workflows, and free people to focus on higher-value activities.
